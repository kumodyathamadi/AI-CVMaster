# AI Resume Analyzer - Frontend

Modern React application for resume upload and AI analysis.

## Setup

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run development server:**
    ```bash
    npm run dev
    ```

## Features

- **Drag-and-Drop Upload**: Easy resume upload with visual feedback.
- **Real-time Validation**: Client-side validation for PDF format and file size.
- **Glassmorphism UI**: Premium, modern design with smooth animations.
- **Axios Integration**: Robust API communication with the Flask backend.
